# 🚀 9001APP v7

Sistema de Gestión ISO 9001 - Versión Next.js 14

## 📦 Instalación

```bash
npm install
```

## 🛠️ Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
